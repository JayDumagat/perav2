# perav2

Modern web-based trading platform prototype with:

- Authentication flow (login, registration, onboarding)
- Dashboard for total value, gain/loss, allocation, watchlist, transactions, and quick actions
- Trading page with search, stock detail metrics, interactive time-range chart, and market/limit order ticket
- PERA dashboard with contributions, tax visualization, allocation management, recurring contribution setup, and retirement projection
- Learning hub with articles, learning paths, and quiz/progress interactions
- Light and dark mode toggle
- Responsive corporate flat UI (no rounded corners, no shadows)
- Simulated near real-time watchlist updates

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```
